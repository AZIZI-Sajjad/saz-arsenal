## Hash Table
```
Le Hash Table ou table de calcul est une syntaxe utilisée, notamment dans le cadre des propriétés calculées pour modifier l'affichage d'une ou plusieurs propriétés d'un objet. Il se présente sous cette forme : @{ }
```

## Langage procédural
```
Le langage procédural est basé sur l'exécution de procédures de manière séquentielle (étape par étape) contrairement au langage objet qui est axé sur les données et les actions.
```

## Boucle
```
C'est l'action de répéter une action un nombre défini de fois (dans le cas d'un Foreach) ou de manière indéterminé jusqu'a la validation d'un résultat (dans le cas d'un While)
```

## By Value

## Variable
```
Une variable est un nom associé à une valeur qui sera stockée en RAM, permettant des accès plus rapides qu'une exécution de commande. Les variables sont indispensables en scripting. Une variable peut changer de valeur.
```

## UAC
```
Le User Account Control a été implémenté avec Windows Vista et permet de restreindre les privilèves dans l'environnement d'un utilisateurs, limitant ainsi les risques de compromission d'une machine.
```

## Bloc Param
```
Le Bloc param, positionné en début de script permet de déclarer des paramètres qui seront utilisables comme des options lors de l'appel de la fonction
```

## Fonction
```
Portion de code Informatique englobé dans une structure de fonction qui permettra de rappeler cette portion de code à de multiples reprises, limitant les redites dans le code.
```

## Encodage
```
L'encodage définit la manière dont les caractère sont retranscrits en "langage machine". L'ANSI est défini sur 1 octet permettant d'encoder les caractères standard, contrairement à l'UTF8 qui est encodé entre 1 et 4 octets permettant d'encoder tous les caractères
```

## WS-Man
```
Web Service Management est un protocole défini par le DTMF. Ce protocole standard est utilisé pour l'administration des serveurs et des équipements réseau.
```

## WinRM
```
C'est l'implémentation Microsoft du protocole WS-Management. WINdows Remote Management est présent sur les environnements Microsoft sous forme de service.
```

## Try Catch
```
C'est une structure dédiée à la gestion d'erreurs. Elle permet de capter une erreur de proposer un traitement autre que l'affichage d'un pavé rouge.
```

## Collection

## Enumération

## Constante
```
Une constante est, à l'image d'une variable, un nom associé à une valeur. La différence majeure est que la valeur d'une constante ne peut pas varier. On peut comparer une constante à une variable en lecture seule.
```
